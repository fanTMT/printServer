### **打印设置选项**

```bash
-n num-copies # 指定打印份数 # 示例：lp -n 3 presentation.pdf

-P page-list # 指定要打印的页面列表 # 示例：lp -P 1,3,5-10 doc.pdf # 打印1,3,5-10页 # 示例：lp -P 1-5,8 doc.pdf # 打印1-5页和第8页

-q priority # 指定作业优先级（1-100）# 示例：lp -q 90 urgent.pdf # 高优先级 # 示例：lp -q 10 draft.pdf # 低优先级

```

### **打印机特定选项（-o）**

这些是传递给打印机的具体设置：

```bash

# 封面/分隔页

-o job-sheets=standard # 打印带有作业信息的封面页
-o job-sheets=none # 不打印封面页

# 纸张设置

-o media=size # 指定纸张尺寸 # 示例：lp -o media=A4 doc.pdf # 示例：lp -o media=Letter doc.pdf # 示例：lp -o media=legal doc.pdf # 示例：lp -o media=a5 doc.pdf

# 页面布局

-o number-up=N # 每张纸上打印 N 个页面（支持：1, 2, 4, 6, 9, 16）# 示例：lp -o number-up=2 notes.pdf # 2合1打印

-o orientation-requested=N # 指定页面方向 # 3 = 纵向（Portrait）# 4 = 横向（Landscape）# 示例：lp -o orientation-requested=4 wide.pdf

# 打印质量

-o print-quality=N # 指定打印质量 # 3 = 草稿（Draft）# 4 = 普通（Normal）# 5 = 最佳（Best）# 示例：lp -o print-quality=5 photo.jpg

# 双面打印

-o sides=one-sided # 单面打印（默认）# 示例：lp -o sides=one-sided doc.pdf

-o sides=two-sided-long-edge # 双面打印，纵向（像书本一样翻页）# 示例：lp -o sides=two-sided-long-edge book.pdf

-o sides=two-sided-short-edge # 双面打印，横向（像日历一样翻页）# 示例：lp -o sides=two-sided-short-edge landscape.pdf

```

### **作业标识选项**

```bash

-t title # 指定作业标题（显示在队列中）# 示例：lp -t "Q3 Financial Report" report.pdf

-s # 静默模式，不显示作业ID # 示例：lp -s document.pdf

```

# 远程共享打印服务 功能大纲

## 一、核心打印基础功能

1. 跨设备 / 跨网络打印支持
   - [x] 多终端适配：网页端
   - [x] 跨网络访问：公网远程打印、局域网优先打印
   - [ ] 多协议兼容：支持 IPP（互联网打印协议）、AirPrint（iOS）、Mopria（Android）、WSD 等主流打印协议
2. 打印任务核心操作
   - [x] 文件格式适配：支持 PDF、Word、Excel、图片（JPG/PNG/TIFF）、TXT 等常用格式直接打印
   - [x] 打印参数配置：纸张大小（A4/A3 / 自定义）、打印方向（横版 / 竖版）、份数、单双面、页边距、缩放比例
   - [x] 打印预览：云端预览打印效果，避免错打漏打
   - [ ] 优先级设置：支持高优先级任务插队（如紧急文件）

## 二、设备管理功能

1. 打印机接入与管理
   - [ ] 多打印机支持：添加、删除、分组管理多台打印机（按位置、部门等）
   - [ ] 打印机状态监控：实时显示在线 / 离线、缺纸、缺墨 / 粉、卡纸等异常状态，推送告警通知
   - [ ] 打印机信息配置：设备名称、型号、位置标签、可打印格式 / 纸张限制、最大打印分辨率
   - [ ] 接入方式：云端绑定（通过打印机序列号 / 二维码）、局域网扫描添加、手动输入 IP / 端口添加
2. 权限管理
   - [ ] 打印机权限控制：按用户 / 部门分配打印机使用权限（只读、打印、管理权限）
   - [ ] 访客权限：临时授权外部用户（如客户、合作伙伴）单次 / 短期打印权限，过期自动失效
   - [ ] 管理员权限：设备添加 / 删除、权限分配、参数配置、日志查看等高级操作权限

## 三、打印任务管理功能

1. 任务生命周期管理
   - [x] 任务提交：本地文件上传打印
   - [ ] 任务队列：查看所有打印任务队列，支持暂停、取消、重新提交任务
   - [ ] 任务状态追踪：实时显示「待打印」「打印中」「已完成」「失败」状态，失败自动重试（可配置重试次数）
   - [ ] 历史记录：保存打印任务历史（文件名称、打印时间、页数、份数、使用打印机、操作用户），支持按条件检索（时间、用户、打印机）
2. 任务优化与安全
   - 任务加密传输：打印文件与任务数据全程 HTTPS 加密，防止数据泄露
   - 任务过期清理：未打印任务超时自动删除（可配置超时时间），避免占用资源
   - 重复任务识别：自动识别重复提交的打印任务，提供跳过 / 合并选项

## 四、用户与账号管理功能

1. 账号体系
   - [ ] 多账号类型：个人账号、企业账号（支持部门架构）、管理员账号
   - [ ] 注册 / 登录：手机号 / 邮箱注册、验证码验证、第三方登录（微信 / QQ / 企业微信）
   - [ ] 账号安全：密码加密存储、登录异常提醒、密码重置、账号注销
2. 个人中心
   - [ ] 个人信息管理：昵称、头像、联系方式修改
   - [ ] 打印偏好设置：默认打印机、常用打印参数（纸张大小、单双面等）、通知方式（短信 / APP 推送 / 邮件）
   - [ ] 消费记录（若涉及收费）：查看打印费用明细、充值记录、发票申请

## 五、增值与拓展功能

1. 云端协作打印
   - [ ] 文件共享打印：支持多用户共享打印文件（如团队协作文档），授权他人打印
   - [ ] 远程授权打印：管理员 / 文件所有者远程授权他人打印特定文件，无需传输原文件
   - [ ] 批量打印：支持多文件打包提交，统一配置打印参数
2. 成本控制与统计
   - [ ] 打印计费（企业 / 商用场景）：按页数、彩色 / 黑白、单双面、纸张类型计费，支持阶梯定价
   - [ ] 用量统计：按用户 / 部门 / 打印机统计打印页数、耗材消耗，生成报表（日 / 周 / 月）
   - [ ] 成本管控：设置用户 / 部门打印配额，超额提醒或限制打印
3. 耗材与维护管理
   - [ ] 耗材余量监控：实时显示打印机墨粉 / 墨盒 / 纸张余量，余量不足推送提醒
   - [ ] 耗材采购链接：提供适配耗材的采购入口（企业场景可对接供应商）
   - [ ] 故障报修：打印机故障时，用户可提交报修申请，管理员接收通知并跟踪处理进度
4. 集成与扩展
   - [ ] 第三方集成：支持对接企业 OA、CRM、网盘（百度网盘 / 阿里云盘 / 企业网盘）、邮件系统，实现文件直接打印
   - [ ] API 开放：提供开放 API，支持企业自定义集成或二次开发
   - [ ] 多语言支持：适配中文、英文等多语言界面（适用于跨国企业）

1. 数据安全
   - [ ] 文件加密存储：打印文件云端暂存时加密，任务完成后自动删除（可配置保留时间）
   - [ ] 访问控制：防止未授权用户访问打印机或打印任务数据
   - [ ] 操作日志审计：记录所有关键操作（账号登录、任务提交、权限变更、设备修改），支持日志导出与追溯

   - [ ] 离线打印：支持打印机离线时缓存打印任务，联网后自动执行打印
